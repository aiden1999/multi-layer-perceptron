# Multi-Layer Perceptron

Aim: Predict the Index Flood for a catchment based on the drivers provided.

## Data

The data represents information on 597 UK river catchments.

- Index flood: Median of the annual maximum series of catchment flow in m^3/s

Eight catchment descriptors to predict the index flood

- AREA: Catchment area in km^2
- BFIHOST: Base flow index
- FARL: Flood attenuation due to reservoirs and lakes
- FPEXT: Flood plain extent
- LDP: Longest drainage path
- PROPWET: Proportion of wet days
- RMED-1D: Median annual maximum 1-day rainfall
- SAAR: Standard annual average rainfall
